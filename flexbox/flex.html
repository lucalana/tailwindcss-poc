Flex

Essa propriedade vamos usar para definir como a tag vai se comportar dentro de uma tag flex crescendo ou encolhendo.
Adicionando a classe flex-[número] com um número definimos esse comportamento e também podemos remover uma tag do flex adicionando a classe flex-none, com o flex none ele vai sempre se manter do mesmo tamanho.
